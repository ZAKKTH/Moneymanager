# 💸 家計簿アプリ

## 🚀 概要
1か月の間で、FlutterとFastAPIを用いた家計簿アプリを開発しました。アプリのコンセプトは既存のアイデアを参考にしましたが、設計や機能、UIについては、AIとの壁打ちを繰り返しながら自ら考案・改善を重ねました。
特に、カテゴリごとの予算使用率をグラフで可視化し、予算の消化状況を一目で把握できる機能を実装しました。これにより、利用者が予算を意識しながら支出を管理しやすくなり、効率的な家計管理を支援するアプリを実現しました

---

## ✨ 主な機能

- 支出登録・編集・削除
- カテゴリの追加・編集
- 月ごとの予算設定
- カテゴリ別予算使用率の可視化
- 月別支出集計
- 円グラフ・棒グラフによる分析

## 作ってみた感想
## 💭 作ってみた感想

本アプリは、初めてFlutterとFastAPIを組み合わせて開発したアプリケーションです。Flutterの状態管理や画面構成の理解に苦戦しましたが、公式ドキュメントやAIとの壁打ちを活用しながら実装を進めました。

また、フロントエンドとバックエンドを分離して設計したことで、REST APIの設計やデータの受け渡しについて理解を深めることができました。この経験を通じて、新しい技術を短期間でキャッチアップし、実際に動作するアプリケーションとして完成させる力を身に付けることができました。
